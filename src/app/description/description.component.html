Parking in Lisbon

This project is being conducted in the course of User Centered Design.

The Goal

We would like to develop an app that not only tells you how long you'll take to get somewhere while driving, plus how long it will take for a parking spot to become available, and if there are none, the nearest area where you can find one.

Why?

This is aimed to reduce the time people spend looking for a parking spot, for people with a driver's licence and a car, possibly people between the ages 18 and 50.
